# 原型
***
# 1.什么是原型
    每个JavaScript对象在创建时, 都会与之关联另一个对象, 这个对象就是原型了

# 2.什么是原型链
    JavaScript对象有一条指向原型对象的链, 这条链就是原型链
    当访问一个对象的属性时, 不仅会在对象上查找, 还会在原型中查找, 直到找到这个属性或者到达原型链的末端位置.


`function Foo(){}`

可以通过prototype属性访问原型, 这个属性只有*函数*才有

`Foo.prototype`

用Foo作为一个构造函数, new一个实例对象foo

`var foo = new Foo();`
